# Maya | Signer

Servicio local de firma digital automática para Maya (Odoo). Permite firmar documentos PDF con certificados digitales o DNIe directamente desde el navegador, sin descargas ni subidas manuales.

## Descripción

**Maya | Signer** es una aplicación de escritorio que:

1. **Descarga PDFs** automáticamente desde Maya.
2. **Firma localmente** con DNIe o certificado digital.
3. **Sube los PDFs firmados** de vuelta a Maya.
4. **Todo sin intervención manual** del usuario.
